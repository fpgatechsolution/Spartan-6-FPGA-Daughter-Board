# Key Features:

 FPGA - XC6SLX9-2TQ144C
 OSCILLATOR – 12MHz
 FLASH – M25P80
 95 free I0
 Input supply 5V DC





This is very low cost board so to use this board you need 5V power supply and JTAG cable which is not provided with board
